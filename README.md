Gym Management System (DBMS Project)
Project Overview

The Gym Management System is a database-driven project developed using MySQL to help a Gym Manager efficiently manage and store all gym-related information.

This project focuses on designing and implementing a relational database to manage:
Members (Customers)
Trainers
Membership Plans
Payments

The system demonstrates core DBMS concepts including database design, normalization, relationships, and SQL operations.

Objective

The main objective of this project is to:
Design a structured database for a gym
Store and manage member and trainer details
Maintain membership and payment records
Perform CRUD operations using SQL
Apply DBMS concepts in a real-world scenario

Technologies Used

Database: MySQL
Language (if connected): SQL / PHP 

Concepts Used:
ER Diagram
Primary Key & Foreign Key
Normalization
Relationships (1:N, N:1)
Constraints

Features
Gym Manager Can:

Add new members
Update member details
Delete member records
Assign trainers
View trainer details
Manage membership plans
Track payments
View complete database records
Joins

Aggregate Functions
